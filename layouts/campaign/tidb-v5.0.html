TiDB 5.0 正式发布

真正服务核心场景的企业级分布式数据库

经过近一年紧锣密鼓的开发和打磨，TiDB 5.0 具备金融行业核心业务场景的服务能力，成为真正服务核心场景的企业级分布式数据库。

TiDB 5.0 性能与稳定性得到显著提升，从而拥有更强大的 OLTP 金融级核心场景服务能力；在原有 HTAP 引擎 TiFlash 基础上引入 MPP 架构，使得众多企业的实时/交互式 BI 成为现实，为高成长企业和数字化创新场景提供了一栈式的数据服务底座，加速带动 HTAP 进入更多大型企业的数字化场景。

TiDB 5.0 新特性
多项性能指标实现突破
通过开启聚簇索引、异步提交事务、MPP 架构提升性能

聚簇索引：开启后，TiDB 在 Sysbench 测试中，Read Write 性能提升达 58.1%;

异步提交事务：开启后，平均延迟由 12.04 ms 降到 7.01ms ，降幅达 41.7%;

MPP 引擎：MPP 引擎总体性能是 Greenplum 6.15.0 与 Apache Spark 3.1.1 的 2-3 倍，部分查询可达 8 倍。
